# Built-in AI — Demo (Cyber-Blueprint)

Proyecto demo para presentación sobre Built-in AI y Modern CSS Standards.

Quick start:

```bash
npm install
npm run dev
```

Características:
- React + TypeScript + Vite
- Tailwind CSS
- Hook `useBuiltInAi` con detección nativa y simulador de streaming
- CSS Anchor & View Transitions demo-friendly patterns
